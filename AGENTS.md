# scAgent Project Context

This is an scAgent project — an agentic single-cell RNA-seq analysis system for 10x Genomics Chromium data.

## Project Structure

- `.pi/SYSTEM.md` — System prompt (scAgent identity and rules)
- `.pi/skills/` — Analysis skills (QC, clustering, annotation, etc.)
- `.pi/extensions/` — Custom pi extensions
- `tools/` — Tool registry (JSON schemas for each analysis tool)
- `outputs/` — Research documents, architecture docs, reports
- `AGENTS.md` — This file (project context loaded at startup)
